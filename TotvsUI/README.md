## Getting Started

Primeiro, inicie o ambiente de desenvolvimento:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abra [http://localhost:8080](http://localhost:8080) no seu browser para ver o resultado.
